Hello World

このページでは、TypeScriptファイルを作成し、コンパイルしてJavaScriptとして実行する流れを確認します。

学習目標
  • TypeScriptファイルを作成できる
  • tsc でJavaScriptへ変換できる
  • 型エラーが出たときに原因を探せる

最初のTypeScriptを書く

まずは文字列を変数に入れて、コンソールに表示します。TypeScriptでは、値の種類を型として明示できます。

src/index.ts

const message: string = "Hello, TypeScript";

console.log(message);

コード解説

  • const message は、再代入しない変数を作ります。
  • : string は、この変数には文字列が入ることを表します。
  • console.log は、値をターミナルに表示します。

理解度チェック

Q1. : string は何を表していますか。

解答を見る

変数に入る値が文字列であることを表しています。数値など別の型を代入すると、TypeScriptがエラーとして教えてくれます。
